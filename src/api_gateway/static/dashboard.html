✨ Architecture Features

🔐 JWT Authentication
HS256 token-based auth with configurable expiry and rate limiting (60 req/min)
🛡️ Prompt Guard
20+ regex patterns detecting injection, jailbreaks, role hijacking, token smuggling
🔒 PII Filter
Detects & redacts emails, phones, SSNs, credit cards, IPs in both input and output
📦 Semantic Cache
Redis + sentence-transformers embeddings with 95% similarity threshold
🔀 Smart Routing
Routes simple queries to GPT-3.5 ($0.002/1K) and complex ones to GPT-4 ($0.06/1K)
📊 Observability
Prometheus metrics, structured JSON logs, Grafana dashboards, request tracing
